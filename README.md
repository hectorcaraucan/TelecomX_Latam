# TelecomX_Latam
Telecom X - Análisis de evasión de clientes (Churn)  Este proyecto forma parte del desafío "Churn de Clientes" para la empresa ficticia Telecom X. El objetivo principal es identificar, procesar y analizar los factores que influyen en la pérdida de clientes, proporcionando insights estratégicos basados en datos para reducir la tasa de evasión.
